Converts a String to a double.

Simple implementation: handles only numbers with upto 15 digits excluding an optional radix point. 15 digits can be stored in long without overflow.

Does NOT ignore spaces at the beginning. Does allow 1 negative sign (-). Radix point is the dot char (.) and is optional (can convert ints like 123).

Convert all digits in string to number, ignoring radix (12.345 => 12345). Divide by 10 to the power of fractional digits with pow().

There has to be at least 1 digit before radix. If there is radix there has to be at least 1 digit following. The number of digits shall not be more than 15. If the number is not convertible returns 0.0 and sets offset to 0.

Parameters
strThe string to convert.
offsetOffset from start until end of number, incl sign and radix.
