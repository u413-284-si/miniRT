int ft_render_sample ( t_render *  render,
uint8_t *  cur_sample 
)

Raytraces a sample of the scene to the sample buffer.

Checks current sample goal. If it is 1, the scene is raytraced without anti-aliasing. If it is more than 1, the scene is raytraced with anti-aliasing. Puts the image to the window. Calcultaes the elapsed time and saves it to the render struct. Increases the current sample number.

Parameters
renderPointer to render struct.
cur_samplePointer to current sample number.
Returns
int 0 if successful, -1 if not.
