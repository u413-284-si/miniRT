◆ ft_check_lines()

t_err ft_check_lines ( char **  lines,
int *  lsrc_count,
int *  total 
)

Check array of imported lines.

Loop through lines array:

  • Check the type of a line with ft_check_line_type().
  • Increment entity counter with ft_incr_ent_count().
  • If ERROR print line number. After loop check entity count with ft_check_entity_count(). Set the provided ints with the calculated values.
    Parameters
    linesArray of imported lines.
    lsrc_countCount of found entities light.
    totalCount of found entities sphere + plane + cylinder.
    Returns
    t_err ERROR if any subfunction fails.
