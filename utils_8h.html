◆ ft_solve()

float ft_solve ( t_equation *  eq)

Solves the given equation.

If the parameter a = 0, the equation is linear and the sole solution is calculated and stored in d1. Otherwise the equation is quadratic and dependent on the discriminant real solutions exist or not:

Discriminant < 0: no real solution, return -1 Discriminant = 0: one solution, return 0 Discriminant > 0: two solutions, return > 0
