◆ ft_quaternion_rot()

t_vec3 ft_quaternion_rot ( t_vec3  vec,
t_vec3  axis,
float  angle 
)

Performs a rotation of a given vector around an axis by a given angle.

The vector is transformed into a quaternion p. A quaternion q is created with the axis and the given angle. The inverse of q is calculated by setting the vector of q_inv as the vector of q scaled by -1. The rotation is effected by multiplying q with p, and thereafter the result with q_inv (lefthand multiplication).

Parameters
vecVector to be rotated
axisAxis around which to rotate
angleAmount of rotation
Returns
t_vec3
