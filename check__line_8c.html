Tries to convert str to float and checks range.

Line gets converted with ft_strtod(). Error if conversion fails. Error if number is out of range. Error if number is not followed by correct separator. Correct separator can be controlled by passed bool: ',' or ' ' and '\0'. If conversion is successful, the line gets moved forward.

Parameters

minMinimum value for converted number.
maxMaximum value for converted number.
commaControls if separator should be ',' or ' ' and '\0'.
Returns
true Float is valid.
false Conversion failed, out of range, wrong separator.

Checks three floats, separated by ',' and calcs checksum.

Works in principle like ft_isvalid_float_block(). Also keeps track of the checksum of all three numbers. If checksum is 0.0, the vector is not a unit vector, which is an error. if checksum is bigger than 1.0, the vector was not normalized. Since it can be normalized, this is only a warning.

Parameters
linePointer to current line.
minMinimum value for converted number.
maxMaximum value for converted number.
Returns
true Unit vector is valid.
false Unit vector is not valid.
