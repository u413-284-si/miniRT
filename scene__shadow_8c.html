Determines whether the point of intersection lies in a shadow regarding the current light source.

Send ray from intersection point towards the light source and check whether it is blocked or not (= shadow ray)

bool ft_shadow_ray_blocked ( t_entities  scene,
t_ray  shadow_ray,
t_interval  ray_d 
)

Determines whether the shadow ray emitted from the point of intersection is blocked on its way to the light source.

An intervall is given to determine the relevant hits min: any value smaller is considered shadow acne (ray hits its own emitting entity due to rounding errors of the float coordinates of the intersection point) max: distance to light source, anything further away is irrelevant

Parameters
sceneScene with all entitites
shadowray Shadow ray in question
ray_dIntervall for relevant hits
Returns
true If shadow ray doesn't reach the light source
false If shadow ray reaches the light source
