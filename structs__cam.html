Detailed Description

Contains camera, image and viewport parameters.

The viewport is a virtual rectangle in the 3D world containing the grid of image pixel locations. If pixels are equispaced horizontally and verti- cally, the viewport has the same aspect_ratio as the image. The focal length is the distance between the camera centre and the viewport centre (vector orthogonal on viewport plane).
