Detailed Description

Pixel grid is a virtual grid of equispaced pixels in the viewport.

The viewport is a virtual rectangle in the 3D world. It contains the grid of image pixel locations. If pixels are equispaced horizontally and verti- cally, the viewport has the same aspect_ratio as the image.

Parameters
delta_uDistance between two pixels in u-direction
delta_vDistance between two pixels in v-direction
pos00Position of the first pixel (0, 0)
